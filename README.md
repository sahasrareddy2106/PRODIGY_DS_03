# 📊 Prodigy InfoTech Internship – Task 3  
### Data Analysis & Visualization

---

## 📌 Task Objective  
The objective of this task is to perform data analysis and visualization to extract meaningful insights from the dataset. This involves understanding patterns, trends, and relationships using various analytical techniques.

---

## 📁 Dataset  
The dataset used in this project is Bank Marketing Dataset, which is analyzed to uncover patterns and insights through visualization techniques.

---

## 🛠 Tools & Libraries Used  
- Python 🐍  
- Jupyter Notebook  
- Pandas – Data manipulation  
- NumPy – Numerical operations  
- Matplotlib – Data visualization  
- Seaborn – Statistical visualization  

---

## 🔎 Data Analysis Steps  
The following steps were performed:

- Data loading and inspection  
- Data cleaning and preprocessing  
- Handling missing values  
- Exploring data distributions  
- Identifying patterns and relationships  

---

## 📊 Visualizations  
Different visualizations were created to understand the dataset:

- Bar charts  
- Histograms  
- Scatter plots  
- Heatmaps  
- Box plots  

These visualizations help in understanding trends and relationships in the dataset.
